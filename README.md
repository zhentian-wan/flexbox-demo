# Flexbox cheetlist


## 1. `display: flex` & `flex-direction`
 
`display: flex` will make align all sibling elements in row;
`flex-direction: column` make elms shows in column

```css
.main-nav ul {
  list-style: none;
  display: flex;
  flex-direction: column; // default : row
}
```
